# Intro til Web componenter i React

Inneheld:

1. Eit overblikk over web komponenter i dag.
1. Demonstrasjon av web komponenter i React.
1. Demonstrasjon av React inni web componenter.

Komme igang:

Før du kan spinne opp dette prosjektet lokalt så må du installere [Node.js](https://nodejs.org/en/) og deretter [Yarn](https://yarnpkg.com/).

1. Kjør `yarn` for å installere avhengigheter.
1. Kjør `yarn dev` for å starte opp en lokal server.
